# Fezzy AI 🤖

A modern AI chatbot powered by Ollama and Flask.

## Features

- Local AI processing with Ollama
- Modern responsive UI
- Fast chat interface
- Dark mode design
- Flask backend
- Privacy focused
- No cloud dependency

## Tech Stack

- Python
- Flask
- Ollama
- HTML
- CSS
- JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/HUSSAM205/Fezzy-AI-Bot.git
cd Fezzy-AI-Bot
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run Ollama:

```bash
ollama serve
```

Run the application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
